#### Overview

- NGINX is an open-source software for:
  - Web serving
  - Reverse proxying
  - Caching
  - Load balancing
  - Media streaming
  - HTTP server capabilities:
    - Proxy server function for email (IMPA, POP3, and SMTP)
    - Reverse proxy and load balancer (HTTP, TCP, UDP servers)
